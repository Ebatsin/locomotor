DEFAULT_ERROR_CODE =(0)

This error code is used when the error message is enough informations to send the client. Wether the message is to be displayed or not to the user is let at the client's discretion.

DISPLAY_MESSAGE =(1)

This error code is used to notify the client that the error message must be displayed.

INVALID_SHORT_TOKEN =(2)

The short token used with the request is not valid (or no longer valid).
